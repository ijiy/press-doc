ESLint格式化 【.eslintrc】 ​

  • eslint convention

ESLint有争议的规则 ​

  • no-magic-numbers, 禁用魔术数字 (建议用常量声明数字, 并注释该数字含义)

  • guard-for-in, 遍历要排除原型链继承来的属性 (很多情况并没必要, 反而影响性能)

js
const obj = {a: 1, b: 2};

for (item in obj) {
	if (obj.hasOwnProperty.call(obj, item)) {
		console.log(item, obj[item]);
	}
}

Object.keys(obj).forEach(item => {
    console.log(item, obj[item]);
});
