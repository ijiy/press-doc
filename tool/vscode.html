markdownlint ​

  • 设置 -> 用户 -> 扩展 -> markdownlint

    • Focus Mode -> 在 settings.json 中编辑

  • markdownlint.config中添加

json
{
	"markdownlint.config": {
		"no-inline-html": false,
		"no-hard-tabs": false,
		"MD004": {
			"style": "sublist"
		},
		"MD007": {
			"indent": 1,
			"start_indent": 4
		},
		"MD046": {
			"style": "fenced"
		}
	},
	"[markdown]": {
		"editor.defaultFormatter": "DavidAnson.vscode-markdownlint"
	}
}
